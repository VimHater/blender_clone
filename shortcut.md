# Shortcuts

## Camera

| Key | Action |
|-----|--------|
| W | Move camera forward |
| S | Move camera backward |
| A | Move camera left |
| D | Move camera right |
| Space | Move camera up |
| Shift | Move camera down |
| Middle mouse drag | Orbit camera |
| Shift + Middle mouse drag | Pan camera |
| Scroll wheel | Zoom in/out |

## Selection

| Key | Action |
|-----|--------|
| Left click (viewport) | Select object |
| Ctrl + Left click (viewport) | Toggle select (multi-select) |
| Ctrl + Left click (hierarchy) | Toggle select (multi-select) |
| Shift + Left click (hierarchy) | Range select |
| Double click (hierarchy) | Rename object |
| Delete | Delete selected objects |

## Transform Gizmo

| Key | Action |
|-----|--------|
| T | Translate mode |
| R | Rotate mode |
| Y | Scale mode |
| Left click + drag gizmo axis | Transform along axis |

## Placement Mode

| Key | Action |
|-----|--------|
| Left click (viewport) | Place object at cursor |
| Escape / Right click | Cancel placement |
